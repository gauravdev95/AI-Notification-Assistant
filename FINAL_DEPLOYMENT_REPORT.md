## 15. Troubleshooting Guide

### If GitHub Actions Build Fails

1. **Check Workflow Logs**
   - Go to: https://github.com/gauravdev95/AI-Notification-Assistant/actions
   - Click failing workflow
   - Expand job logs
   - Identify error message

2. **Common Issues**
   - JDK version mismatch → Fixed (using JDK 17)
   - Gradle cache issues → Clear and rebuild
   - Dependency conflicts → Update versions
   - Kotlin compatibility → Already aligned

3. **How to Fix**
   - Edit code in IDE
   - Commit and push
   - GitHub Actions re-runs automatically

### If APK Download Fails

1. **Try Alternative Locations**
   - GitHub Releases (primary)
   - GitHub Actions artifacts (secondary)
   - Direct download links

2. **Check File Permissions**
   - Ensure read access on GitHub
   - Check download destination

### If Installation Fails

1. **Prerequisites**
   - Android device with USB debugging
   - ADB installed on computer
   - APK from verified source

2. **Installation Command**
   ```bash
   adb install -r app-debug.apk
   ```

3. **Troubleshooting**
   - Ensure device connected: `adb devices`
   - Clear app if reinstalling: `adb uninstall com.gaurav.ainotificationassistant`
   - Check free space on device
